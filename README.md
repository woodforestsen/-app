# 🐴 黑马记账

个人记账应用 — 记录每一笔收支，掌握每一分钱的去向。

---

## ✨ 功能

- **收支双模式**：支持记录支出和收入，一键切换
- **二级分类**：10 个支出大类 + 6 个收入大类，每个大类下有多级小类
- **自定义分类**：可增删改自己的分类，预置分类锁保护
- **明细列表**：按时间倒序展示，支持按类型/分类筛选
- **月度统计**：总收入、总支出、结余一目了然，含分类排行柱状图
- **本地存储**：数据保存在浏览器 localStorage，无需联网，隐私安全

---

## 📸 界面

底部 4 个 Tab 切换不同页面：

| Tab | 功能 |
|-----|------|
| ✏️ 记账 | 添加收支记录，支持支出/收入切换 |
| 📋 明细 | 按时间查看所有记录，支持筛选和删除 |
| 📊 统计 | 月度收支总览 + 分类排行 |
| 🏷️ 分类 | 管理支出/收入的自定义分类 |

## 🚀 运行

```bash
# 安装依赖
npm install

# 开发模式（浏览器打开 http://localhost:5173）
npm run dev

# 生产打包
npm run build

# 预览打包产物
npm run preview
```

```
src/
├── types.ts              # 类型定义、预置分类
├── storage.ts            # localStorage 数据读写

│   └── MonthlyStats.tsx  # 月度统计页面

    └── app.css           # 全局样式
```
